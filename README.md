# Vehicle Physics Plugin

A real-time vehicle dynamics and tire force simulation plugin for Unreal Engine, intended for academic research, education, and interactive simulation and game prototyping.

## Features
- Custom wheel colliders
- Tire force modeling (longitudinal and lateral)
- Drivetrain and powertrain simulation
- Modular architecture suitable for research and experimentation
